# Get Started with Cursor

Use these pages to work with **Red Hat Docs Agent Tools** in **Cursor**. Read [Cursor fundamentals](cursor-fundamentals.md) first, then pick the workflow that matches your project.

- **[Use Cursor with product documentation](cursor-product-documentation.md)** — Your AsciiDoc or Markdown source lives in a separate repository.
- **[Contribute to this repository](../contribute/cursor-contributing-tools.md)** — You are editing plugins, skills, or other files under `plugins/`.
- **[Learn about Cursor workflows](../contribute/cursor-workflows.md)** — You want to learn about the differences between Cursor and Claude Code.
