# Get started with Cursor

Cursor does not use the Claude Code marketplace. You work from a **local clone** of the Red Hat Docs Agent Tools repository and attach skill files to your chat prompts using the `@` context feature.

Read the pages below in order if you are new to using agent tools with Cursor.

- **[Cursor fundamentals](cursor-fundamentals.md)** -- Understand skills, rules, `plugin:skill` naming, and how to load context in Cursor.
- **[Use Cursor with product documentation](cursor-product-documentation.md)** -- Set up a multi-root workspace when your AsciiDoc or Markdown source lives in a separate repository.

## For contributors

The following pages cover contributing to the Agent Tools repository itself from Cursor.

- **[Contributing with Cursor](../contribute/cursor-contributing-tools.md)** -- Edit plugins, skills, or other files under `plugins/`.
- **[Cursor workflows](../contribute/cursor-workflows.md)** -- Learn about the differences between Cursor and Claude Code behavior.
